# AXIS Landing Page

A modern landing page for AXIS, a product and UX audit service for AI-built products that need structure before they scale.

## Files

- `index.html` — main landing page
- `styles.css` — visual design, layout, responsive styles, and theme styling
- `script.js` — light/dark theme toggle

## Deployment

This is a static website. It can be deployed on Vercel, Netlify, Cloudflare Pages, or any static hosting provider.

No build command is required.

## Recommended deploy settings

- Framework: Other / Static HTML
- Build command: leave blank
- Output directory: root `/`
- Entry file: `index.html`
